# HR-Management-System

A comprehensive full-stack Human Resource Management System (HRMS) designed to streamline and automate HR processes for organizations of any size. This system enables efficient management of payroll, employee profiles, time and attendance, performance tracking, and recruitment workflows.

**Key Features:**
- Modular architecture with NestJS (backend) and Next.js (frontend)
- Role-based access control for employees, managers, and HR/admins
- Secure authentication and authorization
- Employee profile management and change request workflows
- Payroll configuration, execution, and tracking
- Leave management with approval workflows
- Performance evaluation and reporting
- Recruitment and onboarding modules
- Organization structure visualization
- Audit trails and activity logs
- RESTful API and modern UI

**Tech Stack:**
- Backend: NestJS, MongoDB
- Frontend: Next.js, React, Tailwind CSS

**Getting Started:**
1. Clone the repository
2. Install dependencies for both backend and frontend
3. Configure environment variables
4. Start backend and frontend servers

For detailed setup and usage instructions, see the documentation in each module folder.
